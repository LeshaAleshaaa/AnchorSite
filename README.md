## Картинки

Папка `images/` — сюда положите логотип, фото для шапки и т.д. Сейчас картинок нет, блоки работают без них. Когда добавите файлы, можно вставить их в HTML (например, `<img src="images/logo.png" alt="Лого">`).

## Как выложить на сервер

1. Подключитесь к хостингу по **FTP/SFTP** или откройте **файловый менеджер** в панели хостинга.
2. Загрузите в **корень сайта** (или в нужную папку для домена) всё содержимое папки:
   - `index.html`
   - `privacy.html`
   - `styles.css`
   - `script.js`
   - `favicon.svg`
   - папку `images/` (если там есть файлы)
3. В панели хостинга привяжите домен к этой папке, если ещё не привязан.
4. Откройте сайт по домену — должна открыться главная страница.

Сборка не нужна: просто заливка файлов.
